Track Package
====================

Easily track a package via UPS, FedEx, USPS, or DHL.

Installation
------------

Download the latest workflow package from the [releases page](https://github.com/Adam81/alfred-track-package/releases) and double click it â€” Alfred will take care of the rest.

Usage
-----

The workflow uses one multiple keyword: "ups", "fedex", "usps", & "dhl".
To look up tracking for a package, type a keyword followed by the tracking code.
